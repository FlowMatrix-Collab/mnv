## Livegang mit GitHub Pages
1. ZIP entpacken.
2. Alle Dateien in ein GitHub Repository hochladen.
3. Settings → Pages öffnen.
4. Deploy from branch → main → root auswählen.
5. Eigene Domain musik-neuverstehen.de eintragen.
6. DNS-Einträge beim Domainanbieter laut GitHub Pages setzen.
7. HTTPS aktivieren.

## Vor Veröffentlichung prüfen
- echte Bilder einsetzen
- Buch-/Verlagslinks ergänzen
- Spotify/Bandcamp/LinkedIn Links ergänzen
- Datenschutz final rechtlich prüfen, wenn externe Dienste eingebunden werden
- Impressum prüfen
- Kontaktformular nur mit externem Dienst ergänzen, da GitHub Pages statisch ist
