## Lokale Vorschau

1. ZIP entpacken.
2. `index.html` oder `VORSCHAU-OEFFNEN.html` öffnen.
3. Die Seite sollte vollständig mit Design/CSS erscheinen.

## GitHub Pages

Wichtig: Lade den INHALT des entpackten Ordners ins Repo hoch, nicht den Ordner selbst.

Im Repository müssen direkt sichtbar sein:

- `index.html`
- `css/`
- `assets/`
- `ueber-mich/`
- `musik-und-entwicklung/`
- `leistungen/`
- `buch/`
- `speaker/`
- `kontakt/`
- `impressum/`
- `datenschutz/`

Dann unter `Settings → Pages` veröffentlichen.
